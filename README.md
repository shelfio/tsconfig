# @shelf/tsconfig

## Install

```
$ yarn add --dev @shelf/tsconfig
```

## Usage

In your `tsconfig.json`

```json
{
  "extends": "@shelf/tsconfig/backend"
}
```

## Publish

```sh
$ git checkout master
$ yarn version
$ yarn publish
$ git push origin master --tags
```
